# REST API Automation Framework (Backend Testing)

Automated tests for a public REST API covering CRUD operations.

## Tech Stack
- Java
- RestAssured (or HTTPClient)
- TestNG
- JSON parsing + assertions

## What This Project Covers
- CRUD API tests (Create, Read, Update, Delete)
- Positive and negative test cases
- Token-based authentication
- Schema validation
- Reusable API utility classes
- Request/Response specifications
